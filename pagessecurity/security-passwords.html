One way that hackers can get your information is by hacking into big company databases. You may hear about some of these on the news. Good news is that you can check if you have been affected by these hacks by going to https://haveibeenpwned.com/ and putting in your email address.

If you use a password on this list, please change it immediately as it is not safe.

If you get a screen like this then you might need to change passwords associated with the listed accounts. And if you reuse the passwords linked to that account then you probably need to change the password on any account that uses it.
